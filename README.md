> Fork a página. Para isso, entrar na página que eu quero e clicar em "Fork"
## 2º Passo) 
> Na sua própria página, pegar a url para clonar no seu computador.

## 3º Passo)
> Você deve clonar o fork que está no seu projeto, não, do projeto
